## ✅ Projeto Lista de Tarefas (To-Do List)

Este projeto é uma aplicação web simples de lista de tarefas, desenvolvida com foco em praticar conceitos fundamentais de JavaScript, HTML e CSS, além de proporcionar uma experiência intuitiva para organização de atividades do dia a dia.

## 🚀 Funcionalidades

- ➕ Adicionar novas tarefas de forma rápida
- 📋 Exibição das tarefas em lista dinâmica
- ❌ Remoção de tarefas com apenas um clique
- 🎯 Interface simples, direta e fácil de usar
- ⚡ Atualização instantânea dos itens na tela

## 🧱 Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨 

## 🎨 Interface e Design

O projeto possui um design minimalista com:

- Tema escuro (dark mode)
- Destaque em cores roxas para identidade visual
- Layout centralizado e organizado
- Botões com efeito hover para melhor interatividade

## 📱 Responsividade

A aplicação foi adaptada para diferentes dispositivos:

- 💻 Desktop 
- 📱 Mobile

## 🎯 Objetivo do Projeto

Este projeto foi criado com o intuito de:

- Praticar manipulação do DOM com JavaScript
- Entender melhor eventos como clique e interação do usuário
- Desenvolver um sistema simples de CRUD (Create e Delete)
- Aprimorar habilidades em responsividade
- Criar um projeto funcional para portfólio

## 🧠 Conceito

Uma lista de tarefas é uma ferramenta essencial para organização pessoal, permitindo registrar, visualizar e gerenciar atividades do dia a dia, ajudando a manter o foco e aumentar a produtividade.
